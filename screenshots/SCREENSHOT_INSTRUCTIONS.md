# Screenshot Instructions

To add a preview image to the README:

1. Open the dashboard in your browser:
   ```bash
   open reports/dashboard_*.html
   ```

2. Take a screenshot of the dashboard (Cmd+Shift+4 on macOS, Print Screen on Windows)

3. Save the screenshot as `dashboard-preview.png` in this `screenshots/` folder

4. The README will automatically show the preview!

## Sample Full Dashboard
